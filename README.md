# Pipeline 🚀

**Pipeline** is a next-generation, automatic job application tracker with AI-assisted tailoring, designed to take the manual data entry out of the job search process.

## Overview

Applying for jobs often means juggling multiple tabs, spreadsheets, and countless cover letter variations. Pipeline solves this by passively tracking your job applications across the web and dynamically generating tailored CVs and cover letters for every role you apply to.

### Key Features

*   **Passive Tracking**: Automatically detects and logs job applications as you submit them across major job boards (LinkedIn, Greenhouse, Lever, Workday, iCIMS, Indeed) with zero manual data entry.
*   **AI-Assisted Tailoring**: Leverages advanced LLMs to map your master CV to new job descriptions. It extracts keywords, aligns your experience, and generates a custom cover letter and tailored CV data.
*   **One-Click Approvals**: The system primes the data for you. You maintain full control by reviewing the AI's suggestions and approving them before anything is sent.
*   **Intelligent Dashboard**: Track your application statuses, interview schedules, and response rates through a beautiful, dark-mode dashboard.

## Built For Speed

The core philosophy of Pipeline is that the user should almost never have to manually type anything into the app. Data enters passively or is generated by AI, requiring only human approval, not authorship.
